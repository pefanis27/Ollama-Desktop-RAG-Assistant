# Ollama-Desktop-RAG-Assistant
Desktop εφαρμογή Python με γραφικό περιβάλλον για σύνδεση με τοπικό **Ollama**, επιλογή **LLM / Embedding** μοντέλων, δημιουργία τοπικών **συλλογών γνώσης (RAG)** και συνομιλία με ή χωρίς RAG.
Η εφαρμογή είναι σχεδιασμένη για χρήστες που θέλουν:
- τοπική χρήση LLM χωρίς cloud εξάρτηση,
- φόρτωση και έλεγχο μοντέλων Ollama,
- επισύναψη αρχείων σε ερωτήσεις,
- δημιουργία knowledge base από τοπικά αρχεία,
- έλεγχο / διόρθωση κώδικα μέσα από ενιαίο GUI,
- ελληνικό περιβάλλον χρήσης και ελληνικές απαντήσεις.

---

## Βασικές δυνατότητες

- Σύνδεση με τοπικό Ollama μέσω HTTP API
- Έλεγχος αν το Ollama app / service είναι ενεργό
- Εκκίνηση Ollama app και Ollama service από την εφαρμογή
- Αυτόματη αναγνώριση διαθέσιμων μοντέλων
- Διαχωρισμός **LLM models** και **Embedding models**
- Επιλογή λειτουργίας:
  - **Συνομιλία χωρίς RAG**
  - **Συνομιλία με RAG**
- Δημιουργία τοπικών συλλογών γνώσης από αρχεία
- Υποστήριξη επισυναπτόμενων αρχείων μέσα στην ερώτηση
- Ειδική χρήση για:
  - έλεγχο κώδικα,
  - διόρθωση κώδικα,
  - εξήγηση αρχείων
- Αποθήκευση απαντήσεων
- Εξαγωγή corrected code από fenced code blocks
- Καταγραφή ενεργειών (logs)
- Σκούρο και φωτεινό θέμα
- Ελληνικό GUI

---

## Υποστηριζόμενα αρχεία

Η εφαρμογή μπορεί να αξιοποιήσει περιεχόμενο από αρχεία όπως:

- `.txt`, `.md`,`.py`,`.json`,`.csv`,`.html`
- `.htm`,`.pdf`,`.docx`,`.js`,`.ts`,`.tsx`
- `.jsx`,`.css`,`.java`,`.cpp`,`.c`,`.h`

---

## Απαιτήσεις

- Python 3.11 ή νεότερη
- Τοπικά εγκατεστημένο **Ollama**
- Windows 10 / 11 (η εφαρμογή μπορεί να λειτουργήσει και αλλού, αλλά έχει σχεδιαστεί κυρίως για Windows desktop χρήση)
